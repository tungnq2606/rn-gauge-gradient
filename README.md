# react-native-rn-gauge-gradient

RN Gauge 

## Installation

```sh
npm install react-native-rn-gauge-gradient
```

## Usage

```js
import GaugeGradient from 'react-native-rn-gauge-gradient';

// ...

<GaugeGradient
    size={200}
    min={0}
    max={100}
    value={70}
    showText
    textValue="70%"
/>
```
